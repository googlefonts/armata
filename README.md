# armata

The sources in `sources/` were converted from this repository's FontForge
`.sfd` sources at commit `fbbc7c2575fe310df0450aaa721ee4e860b03a0e` using [babelfont-rs](https://github.com/simoncozens/babelfont-rs),
and build with [gftools-builder](https://github.com/simoncozens/gftools-builder3) and fontc.

The build was verified to be functionally equivalent to the binary Google Fonts
currently ships for this family.

The original FontForge sources remain in this repository's git history.
